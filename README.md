# apathysadmin
项目后台管理框架
已经完成的功能：
系统管理中用户管理，角色管理，部门管理，菜单管理以及对应的权限分配。
需要开发的功能：
用户登录成功后，缓存用户数据。用户登录具体参考login控制器中的登录方法，现在已经登录成功了，需要继续开发将用户登录信息做缓存处理。

用到的技术有：.netcore 3.1 ,EFCore 5.0,Layui，Sql server 2016 ,Vs 2019
